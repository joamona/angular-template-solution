This is the geoportal that we develop in the subject.

It is a minimum geoportal, with the basics to start, but with all the components.

It has been developed with Django as backend and Angular as frontend.

The map has been built with OpenLayers.

The Django API has been published by using Docker containers

The code of this project can be enlarged with as many components, nd to manage as many OpenLayers interactions as you need.

This is the advantage of using these frameworks.
